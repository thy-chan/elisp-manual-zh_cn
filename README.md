# Emacs Lisp 中文手册

GNU 几乎看不到中文的手册。

为此， 以我这蹩脚的英文水平看英文手册还是很困难的。

创建这个项目主要是想提升自己的英文理解水平， 同时在当中过程可以学习到elisp 的编程技术。

最主要的是个人喜欢用Emacs 编辑器， 不学elisp 怎么能称得上喜欢呢？
